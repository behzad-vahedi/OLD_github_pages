My Projects

GauchoSafe Android App

This project exploit the potential of smartphones as sensors to report crimes or snese of insecurity for UCSB students. Users of the app can enable it when they feel unsafe, and the app will report their location to campus officials or UCSB police department. In case of danger, users will shake their phone, and it will send their location and contact information directly to the police.

Technology Stack: Android, Google Map Api

GauchoSafe Web Interface

This project is a web interface for GauchoSafe app. It provides a heatmap of the locations reported by app users, and police officials can see a detailed report of the locations and also emergency location reports by logging in the website.

Technology Stack: HTML, JavaScript, PHP, Leaflet.js
